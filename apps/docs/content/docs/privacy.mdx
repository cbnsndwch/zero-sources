## Overview

This documentation site uses analytics services to help us understand how visitors use our site and to improve the user experience. This page provides transparency about what data is collected, how it's used, and your rights.

## Analytics Services

### Microsoft Clarity

We use **Microsoft Clarity** to see how you use our website through behavioral metrics, heatmaps, and session replay. This helps us improve our documentation and user experience.

<Callout type="info" title="Data Collection">
Website usage data is captured using first and third-party cookies and other tracking technologies to determine the popularity of content and online activity.
</Callout>

**What Microsoft Clarity Collects:**
- Page views and navigation patterns
- Click interactions and scroll depth
- Device and browser information
- Session recordings (anonymized)
- Mouse movements and heatmaps

**How We Use This Data:**
- Improve documentation content and structure
- Identify usability issues
- Optimize site navigation and search
- Understand popular documentation topics
- Enhance user experience

**Microsoft's Privacy Commitment:**

Microsoft is committed to protecting your privacy. For more information about how Microsoft collects and uses your data, visit the [Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement).

### Vercel Analytics & Speed Insights

We use **Vercel Analytics** and **Vercel Speed Insights** to monitor site performance and understand aggregate usage patterns.

**What Vercel Collects:**
- Page view counts and referrers
- Geographic location (country/region level)
- Device type and browser information
- Page load performance metrics
- Core Web Vitals data

**How We Use This Data:**
- Monitor site performance and availability
- Understand documentation usage patterns
- Optimize page load times
- Identify and fix performance issues

**Vercel's Privacy Commitment:**

Vercel Analytics is privacy-friendly and does not use cookies. It collects anonymous aggregated data and does not track individual users across sites. For more information, visit [Vercel's Privacy Policy](https://vercel.com/legal/privacy-policy).

## Your Privacy Rights

### Data Collection Consent

By using this website, you agree that we and our analytics partners (Microsoft and Vercel) can collect and use your data as described above.

### Opt-Out Options

If you wish to opt out of analytics tracking:

- **Microsoft Clarity**: You can opt out of Clarity tracking by visiting [Microsoft's Opt-Out Page](https://privacy.microsoft.com/en-us/data-privacy-notice)
- **Browser Settings**: Use your browser's "Do Not Track" feature or privacy extensions
- **Ad Blockers**: Many ad blockers also block analytics scripts

### Cookies

This site uses the following types of cookies:

- **Essential Cookies**: Required for site functionality (theme preferences, etc.)
- **Analytics Cookies**: Used by Microsoft Clarity for behavioral analytics
- **Performance Cookies**: Used to measure site performance

## Data Security

We take data security seriously:

- All data transmission uses HTTPS encryption
- Analytics data is processed by reputable third-party services (Microsoft, Vercel)
- We do not sell or share your personal data with third parties for marketing purposes
- We do not collect personally identifiable information beyond what our analytics providers collect
